# ConnectX documentation

This site is built with [Docusaurus](https://docusaurus.io/).

## Prerequisites

- Node.js >= 20
- npm, yarn, pnpm, or bun

## Develop

```bash
cd docs
npm install
npm run start
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run serve
```

## Project layout

- `docs/` — Markdown docs (sidebar is auto-generated from this folder)
- `blog/` — Blog posts
- `src/pages/` — Custom React pages
- `static/img/` — Static assets (add `favicon.ico` and replace `logo.svg` as needed)
